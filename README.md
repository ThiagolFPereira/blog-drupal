# 🔥 Blog Drupal
A blog that talks about animals

## 🔨 Instructions

First Step:
```
- Clone Repository: https://github.com/ThiagolFPereira/blog-drupal.git
```

Second Step:
```
- composer Install
```

### ⚙️ Pre-Requisites

- PHP
- Mysql

* [Drupal] (https://www.drupal.org/)
* [PHP](https://www.php.net/) 
* [Mysql](https://www.mysql.com/)
